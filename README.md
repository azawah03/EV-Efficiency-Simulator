# EV Efficiency Simulator

This project is a Unity-based simulation tool that demonstrates how different parameters affect electric vehicle efficiency.

## Features
- Real-time parameter adjustment using sliders
- Calculation of drag force, rolling resistance, and slope force
- Power consumption and range estimation
- Automatic UI layout system

## Technologies Used
- Unity (C#)
- TextMeshPro
- Unity UI System

## How It Works
Users adjust parameters such as speed, mass, drag coefficient, and slope. The simulator calculates forces and updates outputs in real time.
